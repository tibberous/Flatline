About Flatline

Flatline is a Python process supervisor and crash debugger. Launch any Python application under Flatline and it will monitor its heartbeat, detect freezes, and open an interactive crash console — telling you exactly what the process was doing at the moment of failure.

[FLATLINED TODAY @ 12:07:13.451234 after RESIZE in Flatline::ThemeModal]
Exception: Freeze stalled=3.2s

Always-on supervision. There is no --debug flag. Every launch is monitored, logged, and crash-console-enabled. Flatline selects its heartbeat mode automatically based on whether a database connection is available.

Project Description

Every program contains the seed of its own destruction. A process that freezes under load is not merely a software bug — it is the program's failure to honour the implicit contract it made with the user the moment it started. Most debugging tools make you excavate logs after the fact, like a coroner performing an autopsy instead of a physician keeping the patient alive.

Flatline is built on a different premise: the debugger must be present at the moment of failure. By requiring the supervised process to emit periodic heartbeat messages — named pulses with microsecond timestamps and caller context — Flatline knows not just that the process failed, but exactly what it was doing in the instant before failure.

This is the difference between a crash log and a diagnosis. The log tells you the outcome. The FLATLINED report tells you the story. And stories are what engineers need to actually fix things.

Usage

Basic

python flatline.py app.py
python flatline.py                        # reads app from config.ini
python flatline.py script.py X Y         # passes X Y to script.py
python flatline.py -a X Y script.py      # explicit args via -a
python flatline.py --app script.py       # specify app via --app

Adding heartbeats to your app

from source.remote_client import RemoteClient
flatline = RemoteClient()

class MyWindow:
    def __init__(self):
        flatline.beat('INIT', 'MyWindow')

    def resizeEvent(self, event):
        flatline.beat('RESIZE', 'MyWindow')

    def closeEvent(self, event):
        flatline.beat('CLOSE', 'MyWindow')

Heartbeat Modes

Flatline selects its heartbeat mode automatically at launch based on database availability. Both modes produce identical FLATLINED reports.

DB Mode (database.enabled=true AND connection succeeds)

Child app calls beat(reason, caller) via TCP relay. Each beat is stored in MariaDB with microtime timestamp, reason string, and caller class name. The watchdog queries the DB for the most recent row timestamp and compares it to freeze_threshold.

lastHeartbeat = DB timestamp
Freeze: now() - lastHeartbeat > freeze_threshold

Poll Mode (DB disabled or unavailable — default)

proc.poll() runs in an async non-blocking daemon thread every poll_interval seconds. On each successful poll (child alive), lastPolled = microtime.now(). The watchdog compares now() - lastPolled to freeze_threshold.

lastPolled = microtime of last proc.poll() success
Freeze: now() - lastPolled > freeze_threshold

Zero extra dependencies. Completely non-blocking. Works anywhere Python runs.

Reading the FLATLINED line

[FLATLINED TODAY @ 12:07:13.451234 after RESIZE in Flatline::ThemeModal]
PartMeaning
12:07:13.451234Microsecond timestamp of the last successful heartbeat
RESIZEThe reason string passed to beat()
Flatline::ThemeModalThe caller string passed to beat()

The gap between this timestamp and the freeze detection time = how long the process was silent.

API Reference

class Flatline (flatline.py)

Main supervisor. Always-on. Selects DB or poll mode automatically.

MethodDescription
start()Start relay, watchdog, console, REPL threads. Call before launch().
launch(argv)Launch target script as supervised subprocess. Starts poll thread if in poll mode.
wait()Block until child exits. Opens crash console on non-zero exit.
beat(reason, caller)Record a named heartbeat with microtime. Rate-limited to 50ms per reason+caller.
kill()Force-kill child (SIGKILL / taskkill /F).
terminate()Graceful SIGTERM.
restart()Kill + relaunch with same argv.
shutdown()Stop all threads, close handles.
flatlined()Build the FLATLINED crash fingerprint string.
lastPolledfloat — microtime of last successful proc.poll() (poll mode).
lastHeartbeatfloat — microtime of last beat() (DB mode) or poll (poll mode).

class FlatlineConfig (flatline.py)

Reads config.ini. Falls back to safe defaults on missing keys.

class FlatlineDatabase (flatline.py)

Optional MariaDB logger. All methods silent no-op when unavailable.

class RemoteClient (source/remote_client.py)

Runs inside child process. Sends beat() events via TCP relay.

MethodDescription
beat(reason, caller)Send named heartbeat. Rate-limited.
touchProcessLoop(reason)Update process-loop timestamp.
warn(context, error)Non-fatal warning.
die(context, error)Fatal error — triggers crash console.
onFreeze(seconds, phase)Report UI freeze.
